pytis.wrappers – Interfaces to MD/MC codes¶

class pytis.wrappers.CPMDWrapper(binary, fn_template, fns_other)¶

A wrapper that uses CPMD program to move trough the phase space.

Arguments:
binary – The CPMD binary.
fn_template – A template for the CPMD input.
fns_other – A list of files that must also be present in the directory where the move is computed, e.g. pseudopotential files.
create_input(system, init_state, dirname)¶

Create an input file for CPMD.

Arguments:
system – A System instance.
init_state – A state object with the initial state to put into CPMD.
dirname – The directory where the CPMD job has to be carried out.
get_final_state(system, dirname)¶

Get the last state from the CPMD output

Arguments:
system – A System instance.
dirname – The directory where the CPMD job has to be carried out.
move(system, init_state, dirname)¶

Perform a move trough the phase space.

Arguments:
system – A System instance.
init_state – A state object with the initial state to put into CPMD.
dirname – The directory where the CPMD job has to be carried out.
run_cpmd(dirname, prefix)¶

Execute the CPMD binary on in input file with the given prefix.

Arguments:
dirname – The directory with the input.
prefix – The prefix for the input (.inp) and output (.out)

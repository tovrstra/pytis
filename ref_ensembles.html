pytis.ensembles – Statistical ensembles¶

pytis.ensembles.get_temperature(system, state)¶

Return the instantaneous temperature for a given state of the system.

Arguments:
system – An object of the type System.
state – A corresponding State object.
pytis.ensembles.set_boltzmann_velocities(temp, system, state)¶

Assign random velocities based on the Boltzmann distribution.

Arguments:
temp – The temperature in Kelvin.
system – An object of the type System.
state – A corresponding State object.
class pytis.ensembles.NVTAndersen(temp, rate)¶

The Andersen thermostat for the NVT ensemble.

Arguments:
temp – The temperature in Kelvin.
rate – The probability per unit of time slices that the Andersen thermostat updates the velocities.
update(system, state)¶

Update the velocities (with a certain probability)

Arguments:
system – An object of the type System.
state – A corresponding State object.

This should be called before every time move.

class pytis.ensembles.NVE¶

The NVE ensemble.

This ensemble does not update velocities.

update(system, state)¶
Stub that does nothing.
